# 星尘抽奖 V1 · 使用说明

## 一、你需要准备什么

1. **一台有摄像头的电脑**（Windows 10/11，手势控制需要摄像头）
2. **安装 Node.js**（如未安装）
   - 下载地址：https://nodejs.org
   - 选择 LTS 版本（如 20.x），安装时一路"下一步"即可
   - 安装完成后**重启电脑**（或重新打开命令行）

## 二、启动步骤

1. **双击 `启动服务器.bat`**
2. 看到 "服务器已启动" 后，打开浏览器访问：
   ```
   http://localhost:9999/gesture-particles/stardust-lottery.html
   ```
3. 首次启动会自动下载 Three.js 和 MediaPipe 库文件（约 15MB），请耐心等待

## 三、操作指南

### 键盘操作
| 按键 | 功能 |
|------|------|
| **← →** | 切换粒子球（22个星座主题） |
| **空格** | 抽取奖品 |
| **R** | 重置所有状态 |
| **A** | 自动连续抽奖模式 |
| **G** | 开启/关闭手势控制 |
| **右上角 ▼** | 展开中奖记录面板 |

### 手势操作（需先按 G 或点击 ✋ 按钮开启）
| 手势 | 功能 |
|------|------|
| **手部移动** | 360° 旋转粒子球，像抓在手里一样 |
| **☝ 仅食指** | 下一个粒子球 |
| **✌ 食指+中指** | 上一个粒子球 |
| **🖐 五指张开** | 粒子发散（张得越大散得越开） |
| **✊ 握拳保持0.4秒** | 爆炸！抽取奖品 |
| **❤ 拇指食指尖相碰** | 爱心特效（80颗爱心飞舞） |

## 四、如何添加/修改奖品

**超级简单，不需要改任何代码！**

1. 打开 `gesture-particles/prizes/` 文件夹
2. 放入文件，按格式命名：**`权重.奖品名称.扩展名`**

### 文字奖品（.txt）
```
5.升职加薪.txt
  └─ 文件内容第一行就是奖品描述文案
```
- 权重 1-4 = 传说(金色特效) | 5-7 = 稀有(蓝色特效) | 8+ = 普通(灰色特效)
- 权重越大越容易中

### 图片奖品（.png / .jpg / .webp）
```
3.豪华大礼.png
  └─ 图片会直接展示在3D球体和弹窗卡片上
```

### 限制抽中次数
```
8.阳光普照_x5.txt
  └─ 最多被抽中5次，用完自动移出奖池
```

3. 刷新浏览器页面，**立即生效！**

更多详情见 `gesture-particles/prizes/奖品配置模板.html`

## 五、常见问题

**Q: 打开页面显示"没有奖品"？**
A: 必须通过 `http://localhost:9999/...` 访问，不能直接双击 HTML 文件。

**Q: 手势控制没反应？**
A: 检查浏览器是否允许摄像头权限（地址栏左侧有摄像头图标）。确保光线充足，手掌正对摄像头。

**Q: 页面很卡？**
A: 3D 粒子渲染对显卡有一定要求，可以关闭手势控制（按 G）减轻负载。

**Q: 如何分享给别人？**
A: 把整个文件夹打包成 zip 发给对方，对方解压后双击 `启动服务器.bat` 即可。

**Q: 可以改成其他端口吗？**
A: 用记事本打开 `server.js`，第 7 行 `const PORT = 9999;` 改成其他数字即可。

## 六、停止服务器

在命令行窗口按 **Ctrl + C** 即可停止。关闭窗口也能停止。下次使用时重新双击 `启动服务器.bat`。
